# Data and code availability statement

All code, the synthetic corpus, and the full analysis pipeline that support the
findings of this study are openly available in the project repository at
[[https://github.com/stevenkayitaresteven/The-Longest-Answer-Heuristic]] under the
[[LICENSE — e.g. MIT]] license. A permanently archived snapshot will be deposited
at [[Zenodo / OSF]] upon acceptance and assigned a DOI ([[DOI]]).

The repository contains: the reusable measurement suite (`src/metrics.py`), the
provider-agnostic data-collection harness (`src/collect.py`), the synthetic-corpus
generator (`src/synthesize_demo.py`), the mitigation toolkit (`src/mitigate.py`),
and the end-to-end analysis script (`src/analyze.py`), together with all generated
figures, tables, and `results/results_synthetic.json`.

**On the data underlying the reported numbers.** The quantitative results in the
present version are derived entirely from a clearly labeled *synthetic* corpus
(`data/synthetic_mcq.jsonl`), generated by `src/synthesize_demo.py`, which injects
a controlled bias so the pipeline can be validated end to end. These numbers are
illustrative of the method and are **not** measurements of any commercial model.

No real model-elicited corpus is released with this version because sanctioned API
access to all six studied systems was not available at submission time. The
collection harness is provided so that any researcher with the relevant API keys
can regenerate a real corpus and reproduce every table and figure with a single
command:

```bash
python collect.py --provider <provider> --model <model-id> --n-per-cell 30 \
       --out ../data/corpus_<name>.jsonl
cat ../data/corpus_*.jsonl > ../data/corpus_all.jsonl
python analyze.py --data ../data/corpus_all.jsonl --tag real
```

Any real corpus collected for a future version will be released subject to the
named providers' terms of service, with model outputs redistributed only to the
extent those terms permit.
